## 波宝钱包TronLink安装教程

TronLink是波场生态最流行的钱包。

### 安装方式

**浏览器插件**
1. 访问Chrome网上应用店
2. 搜索"TronLink"
3. 点击安装
4. 创建或导入钱包

**移动端**
1. 在应用商店搜索"TronLink"
2. 下载安装
3. 设置钱包

### 网络配置

TronLink支持：
- 主网（Mainnet）
- Shasta测试网
- Nile测试网

## 波场Tron一键发币教程

在波场链上创建TRC20代币。

### 创建参数

- 代币名称
- 代币符号
- 精度（通常为6或18）
- 发行总量

### 创建步骤

1. 连接TronLink钱包
2. 填写代币参数
3. 支付TRX手续费
4. 等待链上确认

### 费用说明

- 合约部署费用约100-200 TRX
- 需要消耗能量和带宽

## SunSwap做流动性资金池子教程

SunSwap是波场链上最大的DEX。

### 创建流动性池

1. 访问 <https://sunswap.com/>
2. 连接TronLink钱包
3. 选择"流动性" -> "添加"
4. 选择代币对（如TRX-YOUR_TOKEN）
5. 设置初始流动性比例
6. 确认添加

### 初始价格

初始价格 = TRX数量 / 代币数量

### 注意事项

- 确保有足够的TRX和代币
- 首次添加确定初始价格
- 建议池子大于1000 TRX
