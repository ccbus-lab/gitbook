### 1、功能解释

  * check

**持币暴力分红解读**

    * 就是`持币分红`的意思，持有你发行的代币，就可以分红其他代币，例如分USDT、分Doge、分ETH等等

    * 每个钱包获得的分红，取决于该钱包的持仓量。持仓越大，分的越多

### 3、参数说明

连接好钱包后，我们在PandaTool发币页面：<https://ccbus.cc/create-token/hold-others>，填写相应的参数：

![](https://help.pandatool.org/~gitbook/image?url=https%3A%2F%2F1885923539-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FnmLBiMxr5iATgeZGW8in%252Fuploads%252FSoHxdL5UlMYEjCVxbC0v%252F%25E5%258F%2582%25E6%2595%25B0%25E8%25AF%25B4%25E6%2598%258E.png%3Falt%3Dmedia%26token%3Dafb16dfa-3fe9-48a9-b488-0fff9e5b6d1a&width=768&dpr=3&quality=100&sign=9c0ee424&sv=2)

  * check

**代币全称** : 代币的名称信息，如Ethereum（支持中文、英文以及中英混合文字）

  * check

**代币符号** : 也就是代币简称，如ETH。通常就是`看K软件` `薄饼` `钱包`中显示的那个名称

  * check

**发行量** : 代币发行的总供应量,无法增发,固定发行,如果总量过多的话,需要降低精度

  * check

**精度** : 代表币的小数位数，如：0.000001代表精度为6，只能是18

  * check

**收币地址：** 代币创建成功后接受代币的地址（该地址默认是白名单）

  * check

**买入税率** (不需要的部分填0/总比例要小于25%)

    * **营销税率** : 每笔买入都会扣除对应比例代币送进`合约地址`,在**触发阈值** 时会自动卖出换成`USDT`(这取决于池子类型，底池是什么币营销钱包就进什么) 发送到你的营销钱包地址

    * **销毁税率** : 每笔买入都会扣除对应比例代币送进`黑洞地址`,达到销毁的目的

    * **回流税率** : 每笔买入都会扣除对应比例代币送进`合约地址`,在**触发阈值** 时会自动添加流动性,使池子更厚，加池所得LP默认给到营销钱包

    * **分红税率：** 每笔买入都会扣除对应比例代币送进`合约地址`,在有卖单的时候，合约会自动将存储的代币**卖出** 换成USDT(取决于你的分红代币)发放给持币达到`最低分红标准`的用户

  * check

**卖出税率** (不需要的部分不能填空，必须填0，营销税除外，总比例应该小于25%)

    * 这部分跟买入税率解释一样

  * check

**营销钱包：**

    * 白名单地址，可以获得营销税和回流LP。如果底池是BNB池子，就获得BNB。如果底池是USDT，就获得USDT

  * check

**分红最小持币量：**

    * 一个`持币门槛`限制，最少持币达到某个数量，才能获得分红。该数量不能为0，必须大于0。该功能**设置后不可修改** ，所以请确保设置的符合预期。

  * check

**选择交易所：**

    * 不同的链会有不同的交易平台，BSC一般选择薄饼Pancake。选择什么交易所，就必须去那里添加流动性。搞错了会导致机制无法执行，请注意

  * check

**选择底池代币** :

    * 支持USDT、BNB等交易对

  * check

**选择分红代币**

    * 自行选择要分红的代币，将该代币合约地址填入即可。注意，**选择的分红代币必须有BNB的交易对** ，可以正常买卖的。如果该代币流动性过低或者没有BNB交易对，很可能无法分红。因此，一般建议选择流动性好的主流币。

**手动开启交易**

    * **必须选它** : 需要在控制台打开交易开关,才能够交易（与加池子）,并且打开后无法重新关闭

    * 开启交易前：只有白名单可以交易与加池子。开启交易后，任何人都可以

  * check

**税率开关**

    * **必须选它** : 创建代币后手动修改税率, 买卖税率各小于25%

  * check

**黑名单功能**

    * **必须选它** : 能够`添加`和`解除`黑名单。被拉入黑名单的地址将无法卖出代币，也不能转账，该功能慎用


#### 5、控制台说明

当我们成功发行代币后，可进入控制台，对代币的各项功能进行管理。我们连接钱包之后，找到控制台，修改下列功能：

![](https://help.pandatool.org/~gitbook/image?url=https%3A%2F%2F1885923539-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FnmLBiMxr5iATgeZGW8in%252Fuploads%252FBhmp9Kq4QnqR9VFgHWpY%252F%25E6%258E%25A7%25E5%2588%25B6%25E5%258F%25B0.png%3Falt%3Dmedia%26token%3Def9b340f-0b8c-4718-94da-18cf5b7b2580&width=768&dpr=3&quality=100&sign=9ba2fee6&sv=2)

  * check

**权限控制**

    * **转让所有权** : 将合约权限转让给其他人（转移权限之前，记得复制`控制台链接`。新的权限地址必须通过控制台链接，才能进入控制台操作）

    * **放弃所有权** : 将合约权限丢至黑洞，永远不能拿回

  * check

**交易控制**

    * **开启交易** : 没打开之前，用户不可加池子，不能交易。打开后，用户才能交易，开启后不能关闭。注意，没开启交易之前，**不要丢弃权限**

    * **提取合约分红代币** : 将合约地址内遗留的未分发的分红代币提出，给到权限地址

    * **设置黑名单：** 可以批量添加或者移除黑名单，黑名单地址能进不能出（无法转账或卖币）

  * check

**税率控制**

    * **修改税率：** 可分别修改回流、营销、分红、销毁税率，相加小于25%

    * **设置税率白名单：** 白名单交易没有税率，可批量添加或移除

    * **修改营销钱包：** 更改合约的营销钱包地址

    * **设置分红黑名单** ：被拉入黑名单的地址，将无法获得持币分红


### 5、疑问解答

  * check

**合约权限能不能丢？**

    * 先不要急着丢。等你加了池子，开盘交易之后，确定交易税率都没问题，才考虑丢权限的事情

  * check

**黑洞地址、合约地址会参与分红吗？**

    * 不会，默认的黑洞地址（0x00000.....dead）、代币合约地址、流动性资金池地址，均被排除在分红之外

  * check

**为什么钱包里会多出一个Tracker的代币？**

    * Tracker是一个用于记录分红的代币，以便合约进行分红权重和地址的识别。该代币不可转账，不然会造成分红混乱。

  * check

**加池子、撤池子税率的问题**

    * 默认情况下，加池子、撤池子是没税率的。但不同类型的池子，又有区别

    * 如果是BNB的池子，按照BNB加池，有卖税。按照BNB撤池，有买税。按照wBNB加池/撤池，无税（必须方向正确，前面是BNB，后面是代币）

    * 如果是USDT的池子，只要方向正确（前面是USDT，后面是代币），那就没有税

  * check

**权限转移后，新地址怎么进入控制台？**

    * 转移权限之前，需要先复制控制台链接（在控制台上方能看到`复制链接`的按钮）。当权限转移后，新的权限地址使用控制台链接，就可以进入控制台操作
