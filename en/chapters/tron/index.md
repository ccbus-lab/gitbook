## TronLink Wallet Installation Tutorial

TronLink is the most popular wallet in the TRON ecosystem.

### Installation Methods

**Browser Plugin**
1. Visit Chrome Web Store
2. Search "TronLink"
3. Click install
4. Create or import wallet

**Mobile**
1. Search "TronLink" in app store
2. Download and install
3. Set up wallet

### Network Configuration

TronLink supports:
- Mainnet
- Shasta Testnet
- Nile Testnet

## TRON One-Click Token Creation Tutorial

Create TRC20 tokens on TRON chain.

### Creation Parameters

- Token name
- Token symbol
- Decimals (usually 6 or 18)
- Total supply

### Creation Steps

1. Connect TronLink wallet
2. Fill in token parameters
3. Pay TRX fees
4. Wait for on-chain confirmation

### Fee Description

- Contract deployment fee approximately 100-200 TRX
- Requires energy and bandwidth consumption

## SunSwap Create Liquidity Pool Tutorial

SunSwap is the largest DEX on TRON chain.

### Create Liquidity Pool

1. Visit <https://sunswap.com/>
2. Connect TronLink wallet
3. Select "Liquidity" -> "Add"
4. Select token pair (e.g., TRX-YOUR_TOKEN)
5. Set initial liquidity ratio
6. Confirm addition

### Initial Price

Initial price = TRX amount / Token amount

### Important Notes

- Ensure enough TRX and tokens
- First addition determines initial price
- Recommend pool larger than 1000 TRX
