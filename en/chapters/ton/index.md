## Tonkeeper Wallet Installation Tutorial

Tonkeeper is the most popular wallet application in the TON ecosystem.

### Installation Methods

**Mobile**
1. Search "Tonkeeper" in App Store or Google Play
2. Download and install
3. Create or import wallet

**Browser Plugin**
1. Visit Chrome Web Store
2. Search "Tonkeeper"
3. Install plugin

### Security Tips

- Safely store mnemonic phrase
- Never leak private key
- Watch out for phishing websites

## TON Chain One-Click Token Creation Tutorial

Create Jetton tokens on TON chain.

### Jetton Introduction

Jetton is TON chain's token standard, similar to ERC20.

### Creation Parameters

- Token name
- Token symbol
- Decimals (usually 9)
- Total supply
- Icon URL
- Description

### Creation Steps

1. Connect Tonkeeper wallet
2. Fill in token parameters
3. Pay creation fee
4. Wait for on-chain confirmation

## DeDust Add Pool (Create Liquidity) Tutorial

DeDust is one of the main DEXs on TON chain.

### Create Liquidity Pool

1. Visit <https://dedust.io/>
2. Connect wallet
3. Select "Pools" -> "Create"
4. Select token pair
5. Set initial liquidity
6. Confirm creation

### Fee Setting

DeDust supports different fee tiers, select appropriate fee based on token type.

## Ston.fi Add Pool (Create Liquidity) Tutorial

Ston.fi is another major DEX on TON chain.

### Create Liquidity Pool

1. Visit <https://ston.fi/>
2. Connect wallet
3. Enter liquidity page
4. Select token pair
5. Add liquidity

### Advantages

- Low slippage
- Multiple pool types
- Good user experience

## Lock Pool Platform TonRaffles Usage Tutorial

TonRaffles provides LP locking services to increase project credibility.

### Lock Pool Steps

1. Visit TonRaffles platform
2. Connect wallet
3. Select LP to lock
4. Set lock duration
5. Confirm lock

### Benefits of Locking Pool

- Increase investor confidence
- Prevent Rug Pull
- Improve project reputation
