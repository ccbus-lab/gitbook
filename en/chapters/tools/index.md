## Batch Generate BSC/ETH Wallets

Batch create wallet addresses for EVM-compatible chains.

### Feature Description

- Generate multiple wallets at once
- Export private keys and addresses
- Support multiple export formats

### Usage Steps

1. Set quantity to generate
2. Click generate
3. Download wallet file

### Security Tips

- Operate in secure environment
- Safely store private key files
- Never store private keys online

## Token Batch Transfer Tutorial

Batch send tokens to multiple addresses.

### Use Cases

- Token airdrops
- Batch distribution
- Team allocations

### Operation Steps

1. Connect wallet
2. Select token to transfer
3. Input or import address list
4. Set quantity for each address
5. Confirm and execute

### Fee Description

Each transfer requires Gas fee, batch transfers save Gas

## Four.meme Internal Token Creation Tutorial

Issue Meme tokens on Four.meme platform.

### Four.meme Introduction

Four.meme is a Meme token launch platform on BSC chain, similar to Solana's Pump.fun.

### Token Creation Steps

1. Connect wallet
2. Fill in token info
3. Set initial purchase amount
4. Confirm creation

### Important Notes

- Token automatically joins internal market
- Graduates to PancakeSwap when threshold reached

## Contract Security Detection

Detect token contract security.

### Detection Items

- Is contract open source
- Is there honeypot risk
- Is tax rate reasonable
- Are permissions excessive
- Is minting possible

### Usage Method

1. Input contract address
2. Select chain
3. Click detect
4. View detection report
